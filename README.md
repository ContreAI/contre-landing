# Contre AI - Next.js

Production-ready Next.js app with Supabase auth and full SEO optimization.

## Quick Start

```bash
# Install
npm install

# Add Supabase anon key to .env.local
# Get from: https://supabase.com/dashboard/project/qhxevepqohogtenqxrka/settings/api

# Run
npm run dev
```

Open: http://localhost:3000

## Pages

- `/` - Homepage
- `/agents` - Agents
- `/brokerages` - Brokerages  
- `/about` - About
- `/contact` - Contact
- `/privacy` `/terms` `/cookies` - Legal
- `/login` - Login with Google OAuth

## Deploy

```bash
vercel --prod
```
